DESCRIPTION
------------

This module provides the ability to integrate waia in your website.


INSTALLATION
-------------

1. Place the entire waia_integration directory into your Drupal 
    "sites/all/modules/" directory.
2. Enable the waia_integration module by navigating to: 
    "administer > modules"


REQUIREMENTS
-------------

The extensions php_curl is required to check if javascript file exist.


FEATURES
---------

  * The configuration page is located under 
      "admin/config/services/waia_integration"
